[WHAT IS EVERYTALE]

To clarify in case you're not already familiar: Everytale is an online multiplayer Undertale fangame that combines social game elements (i.e. VRChat) with Undertale style turn-based combat for PvE and PvP. Players can hang out, role-play, chat, and battle in an Undertale/Deltarune style world, together! The game features built-in mod tools so players can import/create their own custom characters to play and fight as, all completely cross-platform.
